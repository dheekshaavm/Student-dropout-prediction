# 🎓 Student Dropout Prediction System

## 📌 Overview

This project aims to identify students who are at risk of dropping out using machine learning techniques.
The system analyzes student data and predicts the likelihood of dropout, enabling early intervention.

---

## 🚨 Problem Statement

Student dropout is a major issue in educational institutions.
Early identification of at-risk students can help improve retention rates and provide timely support.

---

## 💡 Solution

This system:

* Processes student academic and demographic data
* Trains machine learning models to predict dropout risk
* Provides predictions through a backend API
* Can be integrated with applications (web/mobile) for real-time monitoring

---

## 🧠 Machine Learning Models Used

* Logistic Regression
* Support Vector Machine (SVM)
* Random Forest
* XGBoost
* K-Nearest Neighbors (KNN)
* Isolation Forest

---

## ⚙️ Tech Stack

* **Programming:** Python, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **ML Libraries:** Scikit-learn, Pandas, NumPy

---

## 🔄 System Architecture

1. Data Collection (Student dataset)
2. Data Preprocessing (cleaning, encoding, normalization)
3. Model Training & Evaluation
4. Prediction API (Node.js backend)
5. Integration with application

## ✨ Features

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Student-dropout-prediction.git
cd Student-dropout-prediction
```

### 2. Backend setup

```bash
cd backend
npm install
npm start
```

### 3. Run ML model

```bash
cd model
python train.py
```
